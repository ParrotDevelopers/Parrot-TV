How to download Parrot TV for kodi.

1. Install kodi from Kodi.tv

2. Go to settings > System Settings > Add-ons > Unknown sources > Yes

3. Go to settings > File manager > Add Source > Tap on "None" and write "http://parrotrepo.tk" click ok then name it "Parrot Repo"

4. Go to settings > Add-ons > Install from zip file > Parrot Repo > click on plugin.parrot.tv-x.x.x

5. Enjoy It

You can update it by clicking on newer version in "Parrot Repo"
